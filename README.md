# Waypoints for Unix terminal

#### This is a program that is used for fast navigation throught folders in terminal.

---

### Installation
1) Download the code from this repo
2) Execute install.sh
3) Restart your terminal.
4) Done, now you can use it.

---

### How to use it?
***Typing -h after command will give you all information you need about it. <br>
It will work for install.sh and uninstall.sh too***  

If you don't know how to use a function, just don't give it any arguments. <br>
It will tell you its usage.  

If you want to get rid of this program, then run uninstal.sh
